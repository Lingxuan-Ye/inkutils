# Inkutils

## 介绍

小叶的工具集。

## 安装

推荐安装方式：

```shell
rm -r -f ~/inkutils/ && \
git clone https://github.com/Lingxuan-Ye/inkutils.git ~/inkutils/ && \
~/inkutils/deploy.sh
```

本工具集可放置于任意目录下，但应在其顶层目录下 **点击运行** [`deploy.sh`](./deploy.sh) 脚本。
